# Shopify Plus

Shopify's enterprise tier. The differentiators that matter for a senior dev role:

## What Plus gives you that Standard doesn't

- Checkout extensibility surface (more capabilities than non-Plus).
- Shopify Functions (custom backend logic at checkout, discounts, delivery, payment customizations).
- B2B features (companies, catalogs, tiered pricing).
- Multi-store organizations.
- Higher API rate limits.

## 2026 deprecation context

- **Shopify Scripts** — off June 30, 2026. Plus stores must migrate to [[shopify-functions]].
- **Non-Plus checkout scripts** — off Aug 26, 2026 (separate deadline).
- **Checkout.liquid** — being phased out in favor of checkout UI extensions.

# Shopify Senior Interview Prep

## TL;DR

Three deliverables tuned for a senior / middle-plus Shopify role with 1–2 yrs of Shopify experience. Anchored to 2026 platform state: Scripts deprecation deadline (June 30, 2026), non-Plus checkout scripts off (Aug 26, 2026), 2026-04 API changes, [[hydrogen]] on React Router v7. The seniority signal at Shopify is migration thinking + tradeoff honesty.

## Deliverables

- `shopify-study-guide.md` — concepts, APIs, gotchas across themes/[[liquid]], apps/APIs, [[hydrogen]], checkout/[[shopify-functions]]. Includes the 2026 deadlines and recent API changes.
- `shopify-qa-practice.md` — 50 questions with sample answers in a strong-senior voice. Practice 2 min/answer out loud.
- `shopify-system-design.md` — six whiteboard scenarios: B2B tiered pricing, flash sale, Hydrogen with multi-source data, custom bundles via Functions, subscriptions, Plus migration off Scripts/checkout.liquid.

## Key 2026 platform facts

- **Scripts deprecation:** June 30, 2026.
- **Non-Plus checkout scripts off:** Aug 26, 2026.
- **API:** 2026-04 release brought breaking changes in Checkout UI Extensions.
- **[[hydrogen]]:** now on React Router v7.

## Senior signal (positioning)

What separates senior from mid in Shopify interviews: **migration thinking and tradeoff honesty**. Lean into "when *not* to use [[hydrogen]] / [[shopify-functions]] / headless" — that scores well. A junior pitches "use the new shiny thing"; a senior pitches "here's the cost and the smaller migration unit."

## Suggested study order (per the session)

1. Study guide (1 sit).
2. Half the Q&A in front of a mirror.
3. One system design scenario.
4. Other half of Q&A.
5. Remaining scenarios.
